Foothill TreeWhiz - Visualize your Trees

On this page you can visualize the trees you produce. Simply (implement and) use a to_json() method for JSON serialization and copy/paste the output into the green text-input window at the bottom. Alternately, you can simply paste the output of your display() method which outputs the tree in LTN format. Your tree should be beautifully rendered in the panel just below this paragraph.

Try out your own JSON or LTN tree

I've produced the visualization using Mike Bostock's D3 library. Uses the Reingold-Tilford algorithm for layout calculations.
